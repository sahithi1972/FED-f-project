🌟 Features

Input ingredients you already have.

Get recipe suggestions instantly.

Recipes come with images, cooking steps, and details.

Helps in reducing food waste and saving money.

User-friendly web interface.

🚀 How It Works

User enters the available ingredients into the website.

The system matches these ingredients with recipes.

Suggestions are displayed with recipe images and cooking details.

Users can cook with confidence, reduce waste, and save resources.


🎯 Aim of the Project

To encourage sustainable cooking habits by reducing food waste, supporting smart usage of ingredients, and promoting eco-friendly lifestyles.
